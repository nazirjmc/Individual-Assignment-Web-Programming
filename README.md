## Project Description

This project is a personal portfolio website built from scratch using **HTML5** and **CSS** only.  
It showcases:

- My demographic information and profile
- A résumé-style professional section
- My personality traits, hobbies, and personal motto
- A reflection on an AI business idea from the “AI Businesses of 2025” blog
- Personal development reflections based on *Twelve Pillars* (Chapters 4–7)
- Storyboards and design decisions for mobile and desktop layouts

## Technologies Used

- HTML5 (semantic tags)
- CSS3 (variables, Flexbox, Grid, responsive design)
- No JavaScript, frameworks, or WYSIWYG editors
